PA5 - Snakes & Ladders

This is a simple Snakes & Ladders game created using the Simple and Fast Multimedia Library (SFML). 

On a Linux system the game can be compiled with once SFML has been installed using the command:

sudo apt-get install libsfml-dev

Once SFML has been installed the "make" command can be used to compile the game. Once the game has been compiled it can be started by running the command "./sfml-app".

Once the game is running Player 1 moves his piece by pressing "r" and Player 2 moves his piece by pressing "p". The first player to reach the 100th square wins the game, and the game closes shortly after.
